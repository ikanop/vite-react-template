## Installation

### Option 1: Using Git Clone

```bash
git clone https://github.com/ikanop/vite-react-template.git
cd vite-react-template
npm install
```

### Option 2: Using Degit

```bash
npx degit ikanop/vite-react-template my-app
cd my-app
npm install
```
